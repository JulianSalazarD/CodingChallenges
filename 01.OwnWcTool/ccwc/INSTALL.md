# Installation

## Requirements

- [Rust](https://www.rust-lang.org/tools/install) (includes `cargo`)

## Build

```bash
cargo build --release
```

The binary will be available at `./target/release/ccwc`.

## Install globally

```bash
cargo install --path .
```

After installing, `ccwc` will be available system-wide.

## Usage

```bash
ccwc [OPTIONS] [FILE]
```
